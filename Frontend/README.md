# Recruitment Management Dashboard - Frontend

React (Vite) client for the recruitment management dashboard.

## Tech Stack
- React 18 + Vite
- React Router
- Tailwind CSS
- Axios

## Setup

1. Install dependencies
```
npm install
```

2. Create a `.env` file:
```
VITE_API_URL=http://localhost:5000/api
```
For production, point this to your deployed backend URL.

3. Run locally
```
npm run dev
```

4. Build for production
```
npm run build
```

## Deployment
Deploy on Vercel/Netlify. Set the `VITE_API_URL` environment variable in the hosting dashboard to your live backend URL.
